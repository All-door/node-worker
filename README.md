# ALL-DOOR Service Worker

## 설명
**ALL-DOOR Service Worker**

  공간 공유 플랫폼 All-Door Worker로서, 주기적으로 디바이스 연결 확인후 소유자에게 알람 혹은 문자 메시지를 보내는 Service Worker입니다.

## 필요 패키지
* node v4.x
* babel 6.x + ES2015 + ES7(stage-3)
* mongodb
* ioredis
* colors

## 사용 방법
`npm install`

`npm start`
